# NIST Cybersecurity Framework (CSF)

## What It Is
The NIST CSF is a widely adopted framework for understanding, managing, and reducing cybersecurity risk. It organizes security work into five core functions.

## Why It Matters
- **Universal language** for execs and engineers
- **Risk‑based** prioritization and road‑mapping
- **Scalable** for SMBs to enterprises/MSPs

## The Five Core Functions
1. **Identify** — Assets, risks, business context  
2. **Protect** — Access control, awareness, hardening  
3. **Detect** — Monitoring, analytics, alerting  
4. **Respond** — IR planning, comms, containment  
5. **Recover** — Restore services, learn, improve

## How to Use It (Practical)
- **Baseline** your current posture with the 5 functions  
- **Map** projects/tasks to functions (e.g., backups → Recover)  
- **Prioritize** high‑risk gaps first  
- **Communicate** status to leadership in CSF terms

## Example
**Phishing incident:** Detect (alert), Respond (contain), Protect (improve mail filters), Recover (restore access), Identify (asset/users involved).
